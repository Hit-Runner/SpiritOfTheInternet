# README

## 题目要求：

### 验证码

- 由4位数码管随机显示4个字符
  - 字符从10个阿拉伯数字和26个英文字母当中选取
- 用户通过拨码输入相应的ASCII
  - 可一次由16个拨码输入2个字符，分两次输入完成
  - 也可一次由8个拨码输入1个字符，分四次输入完成
  - 每次输入后按"中"键确认
- 最终输入正确则数码管显示"pass"，输入错误则显示"fail"
  - 字母不区分大小写（老师的意思是显示的时候不区分大小写，验证的时候也不区分，就是输入的ASCII码为65和97都代表a。这部分汇编没有做好）
